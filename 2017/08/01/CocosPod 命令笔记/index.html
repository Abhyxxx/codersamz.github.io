pod 命令

Usage(使用方法):

$ pod COMMAND

  CocoaPods, Cocoa library 包管理器.

Commands(命令集):

+ cache         操作CocoaPods缓存
+ deintegrate   从你的项目中整合CocoaPods
+ env           显示 pod 的运行环境
+ init          在当前目录生成Podfile文件
+ install       根据Podfile.lock中的版本安装项目依赖库
+ ipc           进程间通信
+ lib           开发一个pod(用于封装自己的CocoaPods库)
+ list          列出所有pods(超多的，看得我眼花)
+ outdated      显示过时的项目依赖关系
+ plugins       显示可用的CocoaPods插件
+ repo          管理规范pod库
+ search        搜索pods
+ setup         配置CocoaPods运行环境
+ spec          管理pod库的规范
+ trunk         与CocoaPods API进行交互 (e.g. publishing new specs)
+ try           测试一个pod库
+ update        更新过时的项目依赖关系并创建新的Podfile.lock

Options(可选项):

--silent        什么都不显示
--version       显示CocoaPod
--verbose       显示更多的调试信息
--no-ansi       显示不带ANSI码的输出
--help          显示命令帮助文档

以上基于Google 翻译，哈哈哈哈

带有+号的命令，可以在后面加上--help查看更详细的说明
